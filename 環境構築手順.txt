# ローカルデバッグ方法
〇フロントエンド
  cd app
  npm run dev
  または、VSCodeのローカルデバッガのappを起動

〇バックエンド
  cd server
  npm run dev
  または、VSCodeのローカルデバッガのserverを起動（対応中）
